HOW TO INSTALL
--------------
Just download MICKEY-Setup.exe and run it. That's it.

The installer automatically:
  1. Installs the full MICKEY app
  2. Installs all dependencies for you (Node.js, Python,
     Playwright Chromium browser, Faster-Whisper voice engine)
     — a console window shows live progress, please let it finish
  3. Creates a Desktop shortcut named "MICKEY"
  4. Creates a Start Menu entry
  5. Offers "Run MICKEY" when installation completes

REQUIREMENTS
------------
  • Windows 10 / 11 (64-bit)
  • Internet connection during installation (for dependency downloads)
  • ~1 GB free disk space

FIRST RUN
---------
  1. Launch MICKEY from the desktop shortcut.
  2. Fill the registration form (Name, Email, WhatsApp, Country).
  3. Wait for admin approval — MICKEY unlocks automatically.

TROUBLESHOOTING
---------------
  • If voice or browser automation features are missing, re-run the
    dependency setup:  <install folder>\bootstrap\install-deps.ps1
    (right-click → Run with PowerShell)
  • SmartScreen warning? Click "More info" → "Run anyway"
    (the installer is unsigned).

Enjoy MICKEY!
